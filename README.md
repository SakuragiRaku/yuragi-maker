# 🌊 ゆらぎめーかー

複数の環境音を自分好みにミックスして流せるリラックスツール。

雨、焚き火、カフェなど12種類の環境音を組み合わせて、自分だけのサウンド空間を作れます。

## 機能

- 🎧 12種類の環境音 (雨・雷・焚き火・鳥・波・風・カフェ・電車・虫・時計・川・キーボード)
- 🎚️ 個別ボリューム＋マスターボリューム
- 💾 プリセット保存・管理
- ⏰ スリープタイマー (15分/30分/60分)
- ✨ 背景グラデーションアニメーション

## 使い方

`index.html` をブラウザで開くだけで動きます。サーバー不要。

## 技術

HTML / CSS / JavaScript / Web Audio API / LocalStorage
