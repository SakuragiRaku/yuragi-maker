ゆらぎめーかー 仕様書


============================================================
概要
============================================================

アプリ名:   ゆらぎめーかー
ジャンル:   環境音ミキサー
名前の由来: 1/fゆらぎのリラックス効果 + 音を作る「メーカー」

どんなアプリ?
  複数の環境音を自分好みにミックスして流せるリラックスツール。
  雨、焚き火、カフェなど12種類の環境音を組み合わせて、
  自分だけのサウンド空間を作れる。

誰が使う?
  作業BGMが欲しい人、リラックスしたい人。


============================================================
機能一覧
============================================================

環境音カード (12種類)
  雨 / 雷 / 焚き火 / 鳥 / 波 / 風
  カフェ / 電車 / 虫の声 / 時計 / 川 / キーボード
  カードタップでON/OFF切り替え
  個別ボリュームスライダー

ミキシング
  マスターボリュームスライダー
  複数の音を同時再生

プリセット
  6つのデフォルトプリセット
    雨の日のカフェ / 森の中 / キャンプファイヤー / 海辺 / 集中モード / 夜の静けさ
  ユーザー独自のプリセット保存 (名前付け)
  プリセット削除

スリープタイマー
  15分 / 30分 / 60分で自動停止
  残り時間表示

ビジュアル
  再生中カードのグローアニメーション
  背景のグラデーションアニメーション
  ダークモードデフォルト


============================================================
画面構成
============================================================

ヘッダー
  ロゴ + アプリ名 (グラデーション)
  プリセット保存ボタン

プリセットバー
  横スクロール可能なプリセットタグ一覧

サウンドグリッド
  3列グリッド (スマホでは2列)
  各カード: アイコン + 名前 + ボリュームスライダー

コントロールバー (画面下部固定)
  マスターボリューム
  スリープタイマーボタン + メニュー
  全停止ボタン

プリセット保存モーダル
  名前入力フィールド + 保存/キャンセル


============================================================
技術構成
============================================================

フレームワーク: なし (HTML / CSS / JavaScript)
データ永続化:  LocalStorage
音声:          Web Audio API (audio-engine.js)

ファイル構成
  yuragi-maker/
    docs/
      仕様書.txt       この仕様書
    index.html          メインHTML
    style.css           スタイルシート
    app.js              アプリロジック
    audio-engine.js     環境音エンジン
    presets.js          プリセット定義


============================================================
LocalStorageキー
============================================================

yuragi-presets    ユーザー作成プリセットの配列
yuragi-state      再生中の音、マスターボリューム、選択プリセット
